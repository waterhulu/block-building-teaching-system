## 简单句型

### 介绍作品
- **This is a bridge.** (这是一座桥。)
- **I built a bridge.** (我搭了一座桥。)
- **My bridge is tall.** (我的桥很高。)

### 描述特征
- **It is very strong.** (它很坚固。)
- **The bridge is very long.** (这座桥很长。)
- **Look at my beautiful bridge!** (看我的漂亮桥！)

### 询问
- **Can you build a bridge?** (你会搭桥吗？)
- **Is it strong?** (它坚固吗？)
- **What color is your bridge?** (你的桥是什么颜色？)

### 合作
- **Let's build a bridge together!** (我们一起搭桥吧！)
- **You put, I stack.** (你放，我堆。)
- **We did it!** (我们成功了！)

## 互动对话

### 对话1：介绍作品
> **Teacher**: What did you build?
> **Child**: I built a bridge!
> **Teacher**: It looks very strong! What color is it?
> **Child**: It is blue and yellow.

### 对话2：合作搭建
> **Teacher**: Let's build a big bridge together!
> **Child**: OK! I'll build the pillars.
> **Teacher**: Good idea! I'll build the bridge deck.

### 对话3：解决问题
> **Teacher**: Oh no, the bridge fell down!
> **Child**: Let's try again!
> **Teacher**: How can we make it stronger?
> **Child**: Maybe use more blocks for the pillars!

## 学习建议

1. **实物展示**：准备积木实物，边搭边说
2. **重复练习**：每天复习3-5个词汇
3. **情境应用**：搭建时多说英语
4. **正向鼓励**：多说"Good job!"鼓励孩子
5. **亲子互动**：家长一起参与英语对话
