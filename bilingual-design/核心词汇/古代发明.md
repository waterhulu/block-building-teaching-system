# 古代发明 - 双语词汇表

|------|---------|------|------|
| 发明 | invention | /ɪnˈvenʃn/ | An important invention. |

| 指南针 | compass | /ˈkʌmpəs/ | Use the compass to find north. |

| 印刷术 | printing | /ˈprɪntɪŋ/ | Printing changed the world. |
| 活字 | movable type | /ˈmuːvəbl taɪp/ | Movable type printing. |
| 四大发明 | Four Great Inventions | /fɔː ɡreɪt ɪnˈvenʃnz/ | China's Four Great Inventions. |
| 创新 | innovation | /ˌɪnəˈveɪʃn/ | Innovation is important. |
